Film service.

Endpoints

/films

GET : [Film]

Returns a list of all films.

Parameter Type Description
match String An optional name pattern to match. An asterisk may be used as a wildcard.

/films/?

GET : FilmDetails

Returns expanded information about a specific film.

Parameter Type Description
filmID Integer • The film ID.

Property Type Description

FilmDetails : Film

Represents detailed information about a film.

Property Type Description
actors [Actor] The actors that appeared in the film.
